# My Cars API
**My Cars API** is a simple secure API Service built using :
* Node - *to handle the server stuff.*
* Express - *as the web framework, and for RESTApi.*
* [JWT](https://jwt.io/) - *for authentication.*
* [Mocha](https://mochajs.org/) - *for testing framework.*
* [Chai](http://chaijs.com/) - *for asserting.*
* [Chai-Http](http://chaijs.com/plugins/chai-http/) - *for async/http asserting.*
* Live app hosted on [Heroku](https://www.heroku.com/).

## Try it out:
https://my-cars-api.herokuapp.com/

##  Application features:

* Signup new User.
* Users can login and logout.
* User authentication done using JWT.
* JWT sent on each authenticated request in the header via **x-access-token**.
* Authenticated users can Add/Update/Delete/Get their car collection.
* Use of ES6 features and Promises on Server.
* Unit Testing for APIs done with Mocha/Chai/Chai-Http covering all cases.
